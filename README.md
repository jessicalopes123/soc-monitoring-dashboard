# soc-monitoring-dashboard
Dashboard de monitoramento SOC desenvolvido com HTML, CSS e JavaScript para simulação de alertas e logs de segurança.


## Funcionalidades
- Alertas críticos
- Logs de segurança
- Usuários online
- Dashboard responsivo
- Interface cyberpunk

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo
Praticar desenvolvimento frontend aplicado à área de cibersegurança e monitoramento SOC.
